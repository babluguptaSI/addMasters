Instructions

  1. Select Sport: Click "Load Sports" to see available sports, then select one.
  2. Add Parent Series: This is independent - just provide name (sport ID is auto-filled).
  3. Add Series:
    • Enter series name (sport ID is auto-filled)
    • Click "Load Parent Series" to populate the dropdown with parent series for your sport
    • Select the appropriate parent series
  4. Add Matches:
    • Enter match name
    • Set match date and time (defaults to current date/time)
    • Click "Load Series" to populate the dropdown with series for your sport
    • Select the appropriate series
  5. Add AIML URL:
    • Enter URL without http:// (e.g., "13.202.75.85:9007/MLConversion")
    • Select type: Curation (1) or Conversion (2)
    • Enter client ID and language ID
    • Enter language name for display purposes
